# Codeforces Train

Repositório dedicado à resolução de problemas do **Codeforces**, com foco em evolução de lógica, algoritmos e estrutura de dados usando C++.

---

## Objetivo

* Melhorar habilidades em resolução de problemas
* Aprender estruturas de dados e algoritmos
* Aprimorar raciocínio lógico
* Praticar escrita de código eficiente

---

## Tecnologias

* C++
* STL (Standard Template Library)
